⚡ Step 4 — Make It Live with Apps Script (Auto-Refresh)
15
Open the Apps Script editor
In your spreadsheet: Extensions → Apps Script. This opens a code editor where you can automate the spreadsheet.
16
Add a script to auto-import from a Google Drive CSV
Delete any existing code and paste this. Replace YOUR_FILE_ID with the ID from your CSV file's Google Drive URL:
function importDynastyData() { var fileId = "YOUR_FILE_ID"; // from Drive URL var file = DriveApp.getFileById(fileId); var csv = file.getBlob().getDataAsString(); var rows = Utilities.parseCsv(csv); var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName("legislators"); sheet.clearContents(); sheet.getRange(1, 1, rows.length, rows[0].length).setValues(rows); // Log the refresh time var dashSheet = ss.getSheetByName("dashboard"); dashSheet.getRange("A1").setValue("Last updated: " + new Date().toLocaleString()); } // Run every hour automatically: function createTimeTrigger() { ScriptApp.newTrigger("importDynastyData") .timeBased() .everyHours(1) .create(); }
Click Save (floppy disk icon). Then click Run → select createTimeTrigger once to register the hourly auto-refresh. Grant permissions when prompted.
⚠️
The first time you run any Apps Script function, Google will ask for permissions. Click Advanced → Go to [project name] and allow access. This is normal.
17
Alternative: Use IMPORTDATA for a live web-hosted CSV
If your CSV is hosted online (e.g. a GitHub raw URL or public Google Drive link), you can skip Apps Script entirely. In cell A1 of your legislators sheet:
=IMPORTDATA("https://your-csv-url-here.csv")
Google Sheets will re-fetch this CSV every ~60 minutes automatically. All formulas on the dashboard sheet will recalculate after each fetch.
💡
To force an immediate refresh: delete the formula and retype it, or use File → Settings → Calculation → On change and every minute.

🔗 Step 6 — Connect This HTML Dashboard to Your Google Sheet
21
Publish your Google Sheet as a CSV endpoint
In your Google Sheet: File → Share → Publish to web → select the legislators sheet → choose format Comma-separated values (.csv) → click Publish. Copy the URL shown — it looks like:

https://docs.google.com/spreadsheets/d/SHEET_ID/export?format=csv&gid=SHEET_GID
22
Replace the static DB in this HTML file with a live fetch
Open this HTML file in a text editor. Find the function init() block. Replace the static DB = {...} assignment with:
async function init() { const CSV_URL = "YOUR_PUBLISHED_CSV_URL_HERE"; const resp = await fetch(CSV_URL); const text = await resp.text(); const rows = text.trim().split("\n").map(r => r.split(",")); const headers = rows[0]; DB.p = rows.slice(1).map(r => ({ fn: r[0], n2: r[1], rg: r[2], di: r[3], cl: r[4], dy: r[5] === "TRUE", sx: r[6], tt: +r[7], tr: r[9] ? r[9].split(";") : [] })); // ... rest of your init code buildKPIs(); populateSelects(); doSearch(); }
Now every time someone opens this HTML file, it fetches the latest data from your Google Sheet automatically — no manual updates needed.
⚠️
This requires the HTML file to be served from a web server (not opened as a local file) due to browser CORS rules. Upload it to GitHub Pages, Netlify, or any static host for free.
